# 🌍 ZhasylBonus - Экологический сайт с волонтёрской системой

## Полная функциональность

### ✅ Основные возможности

#### 1️⃣ Загрузка фото и локаций 📸
- Отправка отчётов об экопроблемах с фотографиями
- Определение текущей локации через GPS браузера
- Интерактивная карта для выбора точного местоположения
- Превью фотографий перед отправкой
- Категория срочности (низкая, средняя, высокая, критичная)

#### 2️⃣ Интерактивная карта проблем 🗺️
- Карта Leaflet с маркерами всех проблем
- Фильтрация по типам: загрязнение, вырубка лесов, вода, отходы, другое
- Разные иконки для каждого типа проблемы (💨 🌳 💧 🗑️ ❓)
- Всплывающие окна с полной информацией об отчётах
- Просмотр фото и описания каждой проблемы

#### 3️⃣ Бонусная система 💰
- **Регистрация**: 100 стартовых баллов
- **За отчёты**:
  - Низкая срочность: 10 баллов
  - Средняя срочность: 20 баллов
  - Высокая срочность: 50 баллов
  - Критичная срочность: 100 баллов
- **Награды и обмены**:
  - Зелёный значок: 500 баллов
  - Дерево вознаграждения: 1000 баллов
  - Сертификат эко-героя: 2000 баллов
- **История транзакций**: Полная информация о всех операциях
- **Система уровней**: Отслеживание прогресса пользователя

#### 4️⃣ Система пожертвований ❤️
- 6 волонтёрских организаций:
  - 🌳 Зелёная планета (посадка деревьев)
  - 🌊 Чистая вода (очистка водоёмов)
  - 🌍 Борьба с глобальным потеплением
  - 🦁 Защита дикой природы
  - 🔥 Спасение экосистем
  - 🌱 Молодые экологи
- Прогресс-бары сбора баллов
- Информация о воздействии:
  - 100 баллов = 1 посаженное дерево
  - 50 баллов = 100 литров очищенной воды
  - 200 баллов = 1 тонна углерода компенсирована
  - 150 баллов = Защита среды обитания животных

#### 5️⃣ Волонтёрская программа 🌱 (НОВОЕ!)
**6 доступных миссий:**

1. **Посади 5 деревьев** ⭐ Простая
   - Посадить минимум 5 деревьев
   - Награда: 50 баллов + сертификат
   - Длительность: 2 недели

2. **Очистка парка** ⭐⭐ Средняя
   - Собрать минимум 20 кг отходов
   - Награда: 100 баллов + сертификат
   - Длительность: 1 неделя

3. **Составь экомаршрут** ⭐⭐⭐ Сложная
   - Создать маршрут для экотуризма
   - Награда: 200 баллов + сертификат
   - Длительность: 1 месяц

4. **Обучи друзей** ⭐⭐ Средняя
   - Провести мастер-класс для 10+ человек
   - Награда: 75 баллов + сертификат
   - Длительность: 2 недели

5. **Восстанови реку** ⭐⭐⭐ Сложная
   - Очистить реку или озеро
   - Награда: 250 баллов + сертификат
   - Длительность: 3 недели

6. **Найди нелегальную свалку** ⭐ Простая
   - Обнаружить и задокументировать свалку
   - Награда: 80 баллов + сертификат
   - Длительность: 1 неделя

**Система достижений:**
- 👣 Первый шаг - завершить 1 миссию
- 💚 Волонтёр - завершить 5 миссий
- 🦸 Эко-герой - завершить 10 миссий
- ⭐ Легенда - завершить 25 миссий
- 👥 Командный игрок - привлечь 5 волонтёров
- 🌍 Эколог - заработать 1000 баллов

**Сертификаты:**
- Загруженные сертификаты для каждой выполненной миссии
- Возможность скачать сертификат
- Возможность поделиться результатом в социальных сетях

**Рейтинг волонтёров:**
- Глобальный рейтинг всех волонтёров
- Система уровней: Новичок → Активист → Волонтёр → Эко-герой → Легенда
- Медали за топ-3 (🥇 🥈 🥉)
- Отслеживание статистики: завершённые миссии и рейтинг

## 🚀 Как использовать

### Первый запуск
1. Откройте `index.html` в браузере
2. Нажмите кнопку "Вход"
3. Выберите вкладку "Регистрация"
4. Заполните форму (имя, email, пароль)

### После регистрации получите:
- ✅ 100 стартовых баллов
- ✅ Доступ ко всем функциям
- ✅ Возможность создавать отчёты
- ✅ Возможность выполнять миссии

### Главная страница
- Просмотр общей статистики
- Кнопки быстрого доступа к отчётам и карте
- Информация о возможностях сайта

### Раздел "Карта проблем"
1. Просмотрите все отчёты об экопроблемах
2. Фильтруйте по типам загрязнения
3. Нажимайте на маркеры для подробной информации
4. Смотрите фото и описание каждой проблемы

### Раздел "Отправить отчёт"
1. Введите название проблемы
2. Выберите тип (загрязнение, вода, лес и т.д.)
3. Опишите проблему подробно
4. Загрузите фото
5. Установите локацию (автоматически или вручную)
6. Выберите уровень срочности
7. Отправьте отчёт → получите баллы!

### Раздел "Волонтёры" ⭐ НОВОЕ
1. Просмотрите доступные миссии
2. Фильтруйте по сложности (простые, средние, сложные)
3. Нажимайте "Подробнее" для деталей
4. Примите миссию кнопкой "Принять"
5. После выполнения отметьте как "Завершить"
6. Получите баллы, сертификат и рейтинг!
7. Следите за своими достижениями и сертификатами
8. Посмотрите свой рейтинг среди волонтёров

### Раздел "Мой счёт"
1. Просмотрите количество баллов
2. Смотрите статистику (отчёты, заработки, пожертвования)
3. Получайте награды за баллы
4. Следите за историей транзакций
5. Выходите из аккаунта

### Раздел "Поддержать"
1. Выберите организацию
2. Введите количество баллов
3. Пожертвуйте свои баллы
4. Смотрите прогресс сбора баллов
5. Узнавайте о воздействии ваших пожертвований

## 💾 Хранение данных

Все данные сохраняются в **LocalStorage** браузера:
- `user` - текущий пользователь
- `users` - все зарегистрированные пользователи
- `reports` - все отчёты об экопроблемах

Данные сохраняются автоматически при каждом действии.
